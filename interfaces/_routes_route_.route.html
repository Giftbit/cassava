Optional handle

handle: (evt: RouterEvent) => Promise<RouterResponse | null | void> | RouterResponse | null | void
  • Defined in src/routes/Route.ts:20

Handle and respond to the ProxyEvent.

Return a RouterResponse object or a Promise that resoles to a RouterResponse object to handle the event.

Return null or resolve the Promise with null to not respond and let other Routes handle it. postProcess() will still be called in that case.
